# Metadata

- Название: drawio-skill
- Автор: Травинов Сергей (адаптация Agents365-ai)
- Версия: 1.21.0-corporate.1
- Описание: Extension помогает GigaCode уточнять запрос пользователя через Diagram Intake Agent, создавать и редактировать профессиональные диаграммы в формате `.drawio`, а также экспортировать их в PNG, SVG, PDF и JPG через установленный draw.io Desktop CLI. Поддерживаются архитектурные схемы, flowchart, git-flow / branching strategy timelines, ERD, UML, sequence/class diagrams, C4, сетевые и cloud-диаграммы, ML/DL-схемы, визуализация кода, Terraform, Kubernetes, docker-compose и SQL DDL. В корпоративной сборке отключено использование внешних CDN для брендовых SVG-иконок; вместо них используются локальные draw.io shapes.
- Применимость: Использовать, когда задача требует визуальной схемы с точной структурой, уточнения типа и состава диаграммы, редактируемым `.drawio`-источником, корпоративными стилями, иконками draw.io, точным размещением элементов по timeline/lane, экспортом в изображение или PDF. В корпоративной среде macOS/Windows draw.io Desktop устанавливается через внутренний маркетплейс SberUserSoft: `https://sberusersoft.sigma.sbrf.ru/#search/Draw.io`. Если draw.io установлен нестандартно, путь задается через `DRAWIO_BIN` или config-файл `~/.drawio-skill/config.json` / `%USERPROFILE%\.drawio-skill\config.json`.
- Использование: Установить skill, распаковав ZIP в каталог GigaCode CLI: `mkdir -p ~/.gigacode/skills && unzip drawio-skill-corporate.zip -d ~/.gigacode/skills`. Затем установить draw.io Desktop через SberUserSoft, проверить CLI командой `drawio --version` или полным путем (`/Applications/draw.io.app/Contents/MacOS/draw.io`, `C:\Program Files\draw.io\draw.io.exe`). Для нестандартного пути создать config: `{"drawio_bin": "C:\\Program Files\\draw.io\\draw.io.exe"}`. Вызывать skill естественным запросом к GigaCode, например: "Нарисуй sequence diagram процесса оплаты в draw.io" или "Построй архитектурную схему сервиса и экспортируй в PNG".
- Эксперимент:* Работоспособность подтверждена через формирование нескольких диаграмм, как с полным предварительным описанием процесса, так и без. Эффективность ожидается за счет снижения затрат на ручное формирование диаграмм и возможность создания диаграмм drawio теми сотрудниками, у кого этот skill отсутствует.
